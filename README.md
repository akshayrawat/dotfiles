# README

My dotfiles

## Prerequisites

* `make install_deps`

## Installation

* `make`

## Manually setup dependencies

### Iterm2

Set "General > Preferences > Load..." to the `iterm2` folder in this repo.

## Local extensions

Create and use these files for machine specific setup

* `~/.zshrc.local`
* `~/.gitconfig.local`
