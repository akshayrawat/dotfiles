### Iterm2

Set "General > Preferences > Load..." to the `iterm2` folder in this repo.

### Local environment
* `~/.zshrc.local`
* `~/.gitconfig.local`

### Prerequisites

* `brew install stow`
* `brew install git-delta`
* `brew tap homebrew/cask-fonts && brew install --cask font-fira-code-nerd-font`
